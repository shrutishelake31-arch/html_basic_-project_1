# 🍽️ Shruti's Cafe

**Happy Food, Happy Mood**

Shruti's Cafe is a beginner-friendly restaurant website created using HTML. The project was developed to practice web development fundamentals and build a strong foundation in creating structured web pages.

## 🚀 Features

* Restaurant Homepage
* Food Menu with Categories
* Starters, Main Course, Desserts & Beverages
* Customer Information Form
* Tables, Lists, and Form Elements
* Clean and Organized Layout

## 🛠️ Built With

* HTML5

## 📖 What I Learned

* HTML Document Structure
* Headings and Paragraphs
* Lists and Tables
* Forms and Input Elements
* Radio Buttons and Checkboxes
* Creating Complete Web Pages

## 🎯 Future Enhancements

* Add CSS Styling
* Improve User Interface
* Add Images and Icons
* Make the Website Responsive
* Add JavaScript Features
